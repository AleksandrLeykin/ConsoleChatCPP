3. В реализации проекта использовался объектно-ориентированный подход.
   Было создано два класса:
	- класс чата
	- класс регистрации пользователя + структура пользователя
   Использовались контейнеры std::vector для хранения экземпляров структуры пользователя 
   и контейнер для хранения сообщений.

4. Пользовательские функции:
	- string getValue() (получение значения типа string)
	- void getPersons() (получение списка зарегистрированных пользователей)
	- void startMessengerChat() (основная функция, для взаимодействия работы пользователя с программой)
	- void enterChat() (вход в чат)
	- bool sign_in() (функция реализующая аутентификацию уже зарегистрированного пользователя)
	- void user_selection() (функция отправки личного сообщения)
	- string writeMessage() (функция набора сообщения)
	- void MenuChoice() (функция выбора действия)
